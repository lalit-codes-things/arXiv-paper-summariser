# arXiv Paper Summariser — Platform V14

V14 expands the project from a paper summariser into a research-paper publication copilot. The platform helps users write, review, format, and publish research papers while preserving a transparent, author-in-control workflow.

## V14 Goal

Help researchers and students write, review, and publish research papers with guided publication workflows, conference-ready formatting, reviewer simulation, citation intelligence, novelty analysis, plagiarism checks, and rebuttal support.

## Core Capabilities

| Capability | V14 behavior |
| --- | --- |
| Conference formatting | Converts manuscripts into target venue templates, validates sections, and checks submission constraints. |
| Abstract generation | Drafts structured and unstructured abstracts from manuscript goals, methods, results, and contributions. |
| Related work writing | Builds related-work outlines, clusters citations by theme, and produces author-editable prose. |
| Reviewer simulation | Simulates multiple reviewer personas, generates scores, flags weaknesses, and recommends revisions. |
| Rebuttal generation | Converts reviews into response plans, evidence-backed rebuttals, and revision checklists. |
| Citation suggestions | Finds missing citations, recommends canonical and recent work, and detects unsupported claims. |
| Plagiarism checks | Compares manuscript sections for overlap risk and highlights passages requiring rewrite or quotation. |
| Novelty analysis | Compares the contribution against cited and discovered literature to identify novelty claims and risks. |

## Operating Principles

1. **Author control:** Generated content is draft assistance, never an automatic submission.
2. **Traceability:** Every citation, novelty claim, and reviewer recommendation must include source references or explain uncertainty.
3. **Venue awareness:** Formatting and review criteria adapt to the chosen conference or journal.
4. **Integrity first:** Plagiarism and citation checks prioritize research ethics over convenience.
5. **Iterative publishing:** Users move through drafting, simulation, revision, rebuttal, and final formatting as repeatable loops.
